Responsive Embedded Content Javascript Plugin

embedResponsive.js 1.0

1 - Introduction

A little plain javascript plugin which makes your embedded content (iframe, object, ...) usable for responsive websites in a flexible way. The css and markup basics are taken from an article at smashingmagazine.com (making-embedded-content-work-in-responsive-design)
